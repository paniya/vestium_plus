Class: SceneObject

SceneObject

new SceneObject(in_guid, (Optional), in_cb)

A object to handle and represent a Lagoa SceneObject outside of the platform This object will be initialized – based on the input guid – producing a mirror object locally outside of the embed.

Parameters:
Name Type Description
in_guid string

guid of an object in the scene
(Optional) Object

PropertySet object to populate the SceneObject. This avoids an async call.
in_cb function

(optional) callback is called when object is done initializing. The callback expects an SceneObject.
